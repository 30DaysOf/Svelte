## 001: Install VS Code Extension

 1. Svelte maintains [this](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode) extension for Visual Studio Code that provides syntax highlighting for Svelte files. Install as usual.

 2. Open `Code > Preferences > Settings` and edit the `Text Editor > Default Formatter` option to select Svelte. This only overrides the default formatter to use the Svelte version in files where that extension is active. 

 3. The Svelte extension has customizable settings. Check [the documentation](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode#settings) for information on what to set and why.

 ✨All done!
