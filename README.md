# Svelte #In30Days

Documenting my exploration of [Svelte](https://svelte.dev/) starting with the recently launched [SvelteKit 1.0](https://kit.svelte.dev/). 

Learning Objectives are:
 - understand syntax and semantics of Svelte
 - explore tools & templates support in SvelteKit
 - get familiar with support utilities (including Vite and TurboRepo)
 - build at least one sample or end-to-end application to see concepts in practice
 
Content Objectives are:
 - document daily progress as I go
 - document relevant resources I used, to clarify understanding where needed
 - compile content into a #30Days Roadmap (for others to use) once learning journey is done
